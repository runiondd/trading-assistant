## Key Decisions Made
- **"Scout" branding** — rejected "Copilot" (competitor), chose "Scout" from 5 options
- **Price Context Map replaces TradingView** — user wanted a unified visualization showing all signals relative to price. Bullish right / bearish left layout with tug-of-war was well received ("the price ladder looks incredible")
- **Explain button** — user wanted `?` buttons next to every insight that explain what it means in plain English via Scout AI. Reusable component, streams response into popover.
- **Stop clears on direction change** — user discovered they could carry a long stop into a short trade. Fixed by clearing stop field on direction toggle.
- **Sonnet model** for Scout — cost efficiency (~$0.01-0.02/conversation)
- **No auto-analysis** — user must explicitly click Analyze or ?
